# Expert.Xchange

Expert.Xchange is a **skill exchange web platform** that allows users to learn and teach skills using a **credit-based barter system**, without any monetary transactions. The project promotes collaborative and accessible learning by enabling people to exchange knowledge instead of money.

---

## âœ¨ Key Features

* User authentication (Sign up / Login)
* User profile creation and management
* Skill listing and discovery
* Skill request, accept, and decline system
* Credit-based skill exchange (earn by teaching, spend by learning)
* Admin management for users and credits
* Responsive web interface

## âš™ï¸ Setup Instructions

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/expert-xchange.git
   ```

2. Move the project folder to XAMPP `htdocs`

3. Start **Apache** and **MySQL** from XAMPP Control Panel

4. Create a database in phpMyAdmin (e.g., `expert_xchange`)

5. Import the SQL file from the `db` folder

6. Open the project in browser:

   ```
   http://localhost/expert-xchange/
   ```
